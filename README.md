## My Vim Configs!
This is my personal vim configs git repo.

## Installation
1. clone into your home dir:

        git clone https://github.com/qizhihere/vim ~/.vim

2. install vim plugins:

        sh ~/.vim/install.sh

3. restart vim and enjoy!
